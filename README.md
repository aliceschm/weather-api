# Weather API Test

Simple Python project to test **WeatherAPI**, making a weather forecast request and displaying the formatted information in the terminal.

### Functionality

The script performs:

- Request to WeatherAPI’s *forecast* endpoint  
- Extraction of:
  - Current temperature  
  - Weather condition  
  - Day’s max and min temperatures 
  - Chance of rain  
  - Moon phase  
  - Hour-by-hour temperatures  
- Organized output printed to the terminal

- For more details about the API, visit:  
https://www.weatherapi.com/
