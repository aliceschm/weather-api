# Weather API Test

Projeto simples em Python para testar a **WeatherAPI**, fazendo uma requisição de previsão do tempo e exibindo as informações formatadas no terminal.

## Funcionalidade

O script realiza:

- Chamada ao endpoint de *forecast* da WeatherAPI  
- Extração dos dados:
  - Temperatura atual  
  - Condição do tempo  
  - Máxima e mínima do dia  
  - Chance de chuva  
  - Fase da lua  
  - Temperaturas por hora  
- Impressão organizada no terminal
